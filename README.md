# Boba-Roller
This is a 3D modelled tool to roll tapioca boba pearl batter into small balls. I made this project for the Hack Club Event, Stasis.

I made this becuase sister likes to make bubble tea from scratch, and she makes her tapioca pearls by hand-rolling each pearl, which is very time-consuming and tedious. So, to make her life easier, I decided to design this boba roller (inspired by cakepop rollers) so she can make multiple boba tapioca pearls at a time!
